Swahilipot Hub Foundation Attachees Attendance Management System

Overview
--------
Full-stack attendance management system for Swahilipot Hub Foundation. Features include secure authentication, GPS-based attendance verification, QR-code verification, role-based access control, reporting, analytics, and email notifications.

Repository layout (phase 1 created files)
- backend/sql/schema.sql  : PostgreSQL schema for all core tables
- backend/.env.example   : Environment variable template
